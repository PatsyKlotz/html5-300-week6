Assignment Six: RESTful Data API

Create a web page with separate forms for each of the basic database operations, similar to the ones used in the lecture. Will need forms for creating, reading, updating, and deleting an item. Should also have a button for listing all of the items. For each of the forms, and the button, the event handler should make the appropriate AJAX call. Can use console.log() to display results, but it's better to display the results on the web page.
